## Quick Steps:

1. **Copy your images** to the `seed-weed-album/public/images/seeds/` folder
2. **Rename them** exactly as listed above (case-sensitive!)
3. **Ensure they are in JPG/JPEG format** (the code accepts .jpg)

## Notes:

- Image filenames must match exactly: `cotton.jpg`, `soyabean.jpg`, etc.
- The website will automatically show placeholders if images are missing
- Recommended image size: 800x600px or larger for best quality
- All images should be optimized for web (not too large file sizes)

## Verification:

After placing the images, restart your dev server or refresh the browser to see them in the Seed Detail View when you click on each seed!
